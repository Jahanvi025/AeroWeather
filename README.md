# AeroWeather

AeroWeather is a simple, responsive weather application that provides real-time weather information. This app allows users to view current weather details, including temperature, humidity, wind speed, and more. It fetches weather data from a weather API and displays it in a user-friendly format.

## Features

- Search for weather by city name.
- Displays current temperature, humidity, wind speed, and weather conditions.
- Responsive design that works on both desktop and mobile devices.
- Easy-to-use and clean UI.

## Technologies Used

- HTML
- CSS
- JavaScript (Vanilla)
- Weather API (e.g., OpenWeatherMap, WeatherAPI, or another weather data provider)
- Git and GitHub for version control
